🤖 AI Chatbot Integration Guide

📱 Embed Chatbot Widget

Add this single line to any website to embed the chatbot:

<script src="YOUR_DOMAIN/api/widget"></script>

Example:

<script src="https://your-chatbot-domain.com/api/widget"></script>

⚙️ Admin Interface

Access the admin panel to manage websites and view analytics:

https://your-domain.com/api/admin

💬 Standalone Chat Interface

Direct link to the chat interface for testing:

https://your-domain.com/api/chat-interface

🔌 API Endpoints

Chat with the bot:

POST /api/chat
Body: { "message": "Hello!" }

Start new conversation:

POST /api/chat/new

Crawl new website (Admin):

POST /api/crawl
Body: { "url": "https://example.com", "maxPages": 10 }

Get chatbot status:

GET /api/config

🚀 Quick Start

  1. Deploy your chatbot API
  2. Access /api/admin to crawl your first website
  3. Embed the widget using /api/widget script
  4. Test the integration using /api/chat-interface
